What is HTML?

HTML (Hypertext Markup Language) is the standard language used to create and design webpages. It is used to describe the structure of web pages and content such as text, images, and links using markup tags.

Basic Structure of an HTML Document

The basic structure of an HTML document consists of the following key elements:

  • <!DOCTYPE html> - Defines the document type and version of HTML
  • <html> - The root element that encapsulates all content in the document
  • <head> - Contains metadata like title, character encoding, and links to external resources (CSS, JavaScript)
  • <body> - Contains the visible content of the web page

Example HTML Code

<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to my first webpage!</h1>
    <p>This is an example of basic HTML structure.</p>
  </body>
</html>

Common HTML Tags

Here are some common HTML tags used to structure web pages:

  • <h1> to <h6> - Headings (h1 is the largest, h6 is the smallest)
  • <p> - Paragraphs for blocks of text
  • <a> - Links (used to navigate between pages)
  • <img> - Images (used to embed images on the page)
  • <ul> and <ol> - Unordered (bulleted) and Ordered (numbered) lists
  • <div> - Used for grouping content

Attributes in HTML

HTML elements can have attributes, which provide additional information about the element. Attributes are always written inside the opening tag of an element.

  • href: Specifies the URL of a link (used in the <a> tag)
  • src: Specifies the source of an image (used in the <img> tag)
  • alt: Specifies alternative text for images (used in the <img> tag)
  • class: Specifies a class for styling
  • id: Specifies a unique identifier for an element

Links and Images

To add links and images to your page, use the following HTML tags:

  • <a href="URL">Link Text</a> - Creates a hyperlink
  • <img src="image.jpg" alt="Description"> - Embeds an image
<a href="https://www.example.com">Visit Example Website</a>
<img src="image.jpg" alt="A beautiful scenery">

Lists in HTML

HTML supports different types of lists: unordered lists and ordered lists. Here's how to create them:

  • Unordered List: Uses the <ul> tag and <li> for list items.
  • Ordered List: Uses the <ol> tag and <li> for list items, but the items are numbered.
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

A Note on HTML Formatting

HTML is not case-sensitive, but it's a good practice to write tag names in lowercase. This helps with readability and maintainability.
